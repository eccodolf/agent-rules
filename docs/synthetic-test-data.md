# Synthetic Test Data Generator

## Назначение

Генерация безопасных тестовых данных для LLM.

---

## Требования

- не использовать реальные данные
- генерировать realistic, но fake данные

---

## Примеры

- fake клиенты
- fake транзакции

---

## Использование

- тестирование prompt templates
- regression

---

## Вывод

Synthetic data обязателен для безопасного тестирования.
