# Simulation Environment для LLM

## Назначение

Среда для прогонки сценариев без риска для production.

---

## Примеры

- тест платежей
- retry сценарии
- отказоустойчивость

---

## Использование

- проверка prompt templates
- проверка policy

---

## Вывод

Simulation снижает риск внедрения ошибок.
