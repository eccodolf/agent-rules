# Требования информационной безопасности

## Основные принципы

- Zero Trust
- Least Privilege
- Defense in Depth

---

## Размещение

- Закрытый контур
- Сегментация сети
- Ограничение исходящего трафика

---

## Данные

Запрещено:
- передавать секреты в промптах
- передавать персональные данные без необходимости

Разрешено:
- обезличенные данные
- тестовые данные

---

## Логирование

- Логировать запросы и ответы
- Исключать секреты
- Ограничивать доступ к логам

---

## Угрозы LLM

- Prompt injection
- Data exfiltration
- Jailbreak

### Защита

- фильтрация входа
- ограничение инструкций
- валидация выхода

---

## Секреты

- Использовать secret manager
- Не хранить в коде
- Не передавать в модель
