# Scenario templates

Этот каталог содержит каркасы проектов, которые можно забирать из git вместе с `.ai/`.

Структура:
- `templates/scenarios/django-service`
- `templates/scenarios/fastapi-service`
- `templates/scenarios/proxy-service`
- `templates/scenarios/desktop-app`
- `templates/scenarios/devops-pipeline`

Шаблоны предназначены для bootstrap проекта перед началом работы с Qwen Code.
