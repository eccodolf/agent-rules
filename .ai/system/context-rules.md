# Context rules

## Назначение

Этот документ определяет правила формирования контекста для LLM.

## Основные правила

- Передавать только минимально необходимый контекст.
- Не передавать секреты, токены, ключи.
- Не передавать большие файлы (логи, дампы, бинарные данные).
- Использовать allow-list подход.

## Запрещено

- `.env`, `*.pem`, `*.key`
- дампы БД
- сырые логи с PII

## Практика

- разбивать задачу на шаги
- использовать DSL
- ограничивать размер prompt
