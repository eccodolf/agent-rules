# Database rules

## Назначение

Правила работы с БД при генерации кода.

## Основные требования

- Использовать параметризованные запросы.
- Избегать `SELECT *`.
- Учитывать индексы.
- Писать idempotent операции.

## Безопасность

- Не хардкодить credentials.
- Валидировать входные данные.

## Практика

- Использовать ORM или query builder.
- Разделять read/write.
