# Coding rules

## Назначение

Правила генерации кода агентом.

## Основные требования

- использовать best practices
- писать читаемый код
- избегать дублирования

## Безопасность

- не использовать секреты
- валидировать input
- использовать prepared statements

## Практика

- разделять слои
- использовать service layer
